# Примеры использования Fennec Core

## Демо-проект (examples/demo)

Демонстрационное приложение для проверки компонентов из `src/components` перед публикацией библиотеки.

### Запуск

Из корня репозитория:

```bash
npm run example
```

Или из папки примера:

```bash
cd examples/demo
npm install
npm run dev
```

Приложение откроется по адресу: http://localhost:5174

### Что проверяет демо

- **Форма (Model + Field)** — форма на основе мета-описания: строки, число, чекбокс, дата.
- **Action** — кнопка открывает модальное окно с формой (Model).
- **Collection** — список с фильтрами, сортировкой и пагинацией; данные из мок-API.

Данные и мета подставляются через мок `fetch` в `DemoProviders.jsx`, бэкенд не требуется.
