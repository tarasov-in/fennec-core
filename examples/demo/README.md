# Демо Fennec Core

Проверка компонентов из `src/components` перед публикацией библиотеки.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5174

## Из корня репозитория

```bash
npm run example
```

## Вкладки демо

1. **Форма (Model + Field)** — поля по мета-описанию.
2. **Action** — модальное окно с формой по кнопке.
3. **Collection** — таблица с фильтрами и пагинацией (мок-данные).
